Завдання - №1

  • Варіант № 1 - Буде виводити, але рекомендовано закривати Інструкцію в (;)
  • Варіант № 2 - Буде виводити, так як написано з нового рядка, але рекомендовано закривати Інструкцію в (;)
  • Варіант № 3 - Не буде виводитись в консолі, через те що написано в один рядок та немає (;)
  • Варіант № 4 - Помилка в лапках *, два рядки не будуть виконані тому що (*)

Завдання - №2

  • Варіант № 2.1 - Не буде виводитись, написано все в одному рядку, РІШЕННЯ добавити між інструкціями (;) або просто перенести на другий рядок але рекомендовано (;)
  • Варіант № 2.2 - Вірно
  • Варіант № 2.3 - Не вірно буде виводитись в консоль але константу прийнято обявляти коли нам зрозуміло кінцевий результат (константи)
  • Варіант № 2.4 - Вірно
  • Варіант № 2.5 - Вірно з var

Завдання - №3

  • Варіант № 3.1 - Не вірно, потрібно зворотні дужки (``) а не подвійні
  • Варіант № 3.2 - Не вірно, вертається значення Infinity
  • Варіант № 3.3 - Не вірно, тип даних (undefined) та значення теж саме
  • Варіант № 3.4 - Вірно, працює як перетворювач з (String) в (Number)
